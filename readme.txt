The goal of this plugin is to alleviate the stress caused by website email sending.

## Description
Finally, an email plugin that delivers! Sending emails from your website has never been easier. Simply sign-up, download, install and allow Serasend to handle all of your e-mail needs.

Serasend is an mail sending service (SMTP) that provides an easy way for WordPress users to handle the sending of transactional emails, through the use of WPForms, ContactForms7 or pretty much any plugin that sends mail. 

## Installation

1. Upload the `serasend` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

## Usage

1. Go to the 'Settings' menu in your WordPress admin dashboard.
2. Click on 'Serasend Mail Settings'.
3. Configure your mail settings and click 'Save Settings'.
